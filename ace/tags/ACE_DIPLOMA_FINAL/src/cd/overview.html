ACE - a collaborative editor

The project documentation consists of the following five documents:

  • Project Manual
  • System Requirements
  • User Manual
  • Developers Guide
  • Final Report

Developers Guide

The Developers Guide shows how to build and run the application from source.

User Manual

The User Manual shows a potential user how to use ACE.

Project Manual

The Project Manual contains all project management related information, including work packages, the project plan, and the milestone plan. This document has not been modified at the end of the project. The analysis of the project can be found in the Final Report.

System Requirements

The System Requirements document lists the goals and desired functionality of ACE.

Final Report

The Final Report documents the application ACE, its architecture, design, and implementation. It also introduces the reader to collaborative editing and the concepts of the transformation algorithm. Further it contains an analysis of the whole project.
